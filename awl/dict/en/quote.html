noun
1. A quotation; a statement attributed to a person.
2. A quotation mark.
3. A summary of work to be done with a set price.
4. A price set for a financial security or commodity.
verb
1. To repeat the exact words of (a person).
2. To repeat (the exact words of a person).
3. To prepare a summary of work to be done and set a price.
4. (Commerce) To name the current price, notably of a financial security.
5. To indicate verbally or by equivalent means the start of a quotation.
6. To observe, to take account of.
